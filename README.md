## Development

To get started with development, follow these steps:

1. **Install Bun:**

   ```bash
   curl -fsSL https://bun.sh/install | bash
   ```

2. **Install Dependencies**

   ```bash
   bun install
   ```

3. **Run the Development Server**

   ```bash
   bun run dev
   ```
